Python
--------
It is a popular high-level programming language known for its simplicity. Interpreted Programming Language.
python --->  compilation into byte code   ---> Interpretation -->   machine code(computer can understand)

Extension for python file ---> .py
Command to run a python file ---> python filename

To Print a data at the output
------------------------------
print(data)

To prompt input 
----------------
input()  ---> will return only string. 77 --> "77"

Syntax 
-------
>> Python is case-sensitive programming language (a not equal to A)
>> Indentation matters.

Comments
------------
Single line comments ---> #
multiple line comments ---> '''  '''

Variable 
-----------
A variable is a named memory location used to store data temporarily.

varName=value

Naming rules 
-------------
>> can contain letters, numerics, - _
>> should start with letters, _
>> no reserved keywords

Datatypes
----------
type(obj) --> find the Datatype of an object

Integer ---> Whole Numbers 
Float ---> dec num 

String --> Collection of char wrapped with '' or "".
Concatination --> + . Ex: "Hello"+"World" ---> "HelloWorld"

Ex: "Hello"
Boolean --> True/False
